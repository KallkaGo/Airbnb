# Airbnb
the React practice project

## 1.简介

本项目是仿Airbnb官网的部分页面，使用react+craco +RTK，UI框架采用了MUI，未来可能会引入RTKQ来完善接口的请求。

### 2.食用说明

##### 拉取项目

```
$ git clone https://github.com/KallkaGo/Airbnb.git
```

##### 安装依赖

```
$ npm i  / npm install
```

##### 启动项目

```
$ npm run start
```
